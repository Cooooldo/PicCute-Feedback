PicCute 技术支持

PicCute 是一款面向 macOS 的轻量图片压缩工具：把图片或文件夹拖进来，选个质量，点压缩——几秒内为你减掉约 60% 的体积，画面几乎看不出区别。支持 PNG / JPEG / WebP 互转，亦可批量处理，并提供压缩前后对比预览。本页汇总系统要求、使用说明、常见问题与反馈方式。

关于 PicCute

  • 拖拽即用：打开就会，无需阅读说明
  • 三种格式互转：PNG、JPEG、WebP，包含动态 WebP
  • 批量压缩：整个文件夹扔进去，自动处理子目录
  • 对比预览：压缩前后可拖动滑块对照，确认后再保存
  • 自定义输出：覆盖原文件，或指定输出目录与文件名模板（如 {name}、{ext}）
  • 简单 / 高级两档：一根滑块即可，亦可调质量、速度、抖动、最大色数
  • 可选 oxipng 二次无损：再榨一点体积
  • 体积更大就跳过：绝不帮倒忙
  • 原生体验：完成后系统通知，一键在访达中显示；简体中文 / English 菜单可随时切换

技术栈：Rust + Tauri 2 + React。v0.1.x 名为 PNGCute，自 v0.2.0 起更名为 PicCute。

系统要求

  • macOS 11（Big Sur）或更高版本
  • 支持 Apple Silicon 与 Intel Mac（Universal 构建）
  • 从 Mac App Store 或官方分发渠道安装的应用均适用本页说明

快速开始

  • 将图片或文件夹拖入窗口，选择输出格式与质量后执行压缩
  • 可在设置中覆盖原文件，或指定输出目录与文件名模板
  • 菜单栏 PicCute → Language / 语言 可切换界面语言（与系统语言可独立设置）

常见问题

  • 为什么压缩后体积没有变小？ 与源图已接近最优、或质量设置较高时会出现；可尝试调低质量或检查是否启用了 “若更大则跳过” 等选项。
  • 支持批量与动图吗？ 支持拖入文件夹（含子项）；WebP 含动图时按应用当前版本能力处理。
  • 需要访问我的哪些文件？ 仅处理您通过拖拽或选择明确授权的路径；请在系统与沙盒提示中允许访问相关文件夹（若使用 Mac App Store 版本）。

故障排除

  • 若压缩失败，请确认磁盘空间充足、目标路径可写，且源文件未损坏
  • 若界面无响应，可完全退出后重新打开，并确保系统为支持的 macOS 版本
  • 与某一特定文件必现时，请通过下方反馈渠道附上系统版本、PicCute 版本与复现步骤

PicCute Support

PicCute is a lightweight image compressor for macOS. Drop images or a folder, pick a quality, hit compress — in seconds you’ll trim roughly 60% of the file size with no visible loss. Supports PNG, JPEG, and WebP (including animated WebP) with conversion between them, batch processing, and a side-by-side before/after preview. This page covers system requirements, usage notes, FAQ, and how to reach us.

About PicCute

  • Drag and drop — no manual to read, just open and go
  • Three formats — PNG, JPEG, and WebP, with conversion between them
  • Batch compression — drop a folder and we’ll handle subfolders too
  • Side-by-side preview — drag the compare slider, then save with confidence
  • Custom output — overwrite originals, or pick a folder with a filename template (e.g. {name}, {ext})
  • Simple or Custom mode — one slider does it all, or tune quality, speed, dithering, and max colors
  • Optional oxipng second pass — squeeze out a few more bytes losslessly
  • Auto-skip if the result is larger — never makes a file worse
  • Native feel — macOS notifications, reveal in Finder, and English / 简体中文 UI switcher

Built with Rust + Tauri 2 + React. Originally released as PNGCute (v0.1.x); renamed PicCute from v0.2.0 when JPEG and WebP support landed.

System requirements

  • macOS 11 (Big Sur) or later
  • Apple Silicon and Intel Macs (Universal build)
  • Applies whether you install from the Mac App Store or another official channel

Quick start

  • Drop files or a folder, choose format and quality, then compress
  • Override originals in settings, or set a custom output folder and filename template
  • Use the menu PicCute → Language / 语言 to switch the UI (independently of the system language)

FAQ

  • Why didn’t the file get smaller? Sources may already be well optimized, or quality may be high. Try a lower quality setting or check options such as “skip if larger”.
  • Batch and animation? Folders (including subfolders) are supported. Animated WebP follows the capabilities of your installed PicCute version.
  • Which files are accessed? Only files and folders you explicitly add or select. Grant folder access if macOS or the App Store sandbox prompts you to.

Troubleshooting

  • If compression fails, check free disk space, that the destination is writable, and that the source file isn’t corrupted
  • If the UI freezes, quit fully and relaunch; confirm your macOS version is supported
  • For a reproducible issue, include your macOS version, PicCute version, and steps when you contact us below
